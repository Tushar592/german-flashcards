User guide

How to use Deutsche Vokabeltrainer

Choose a deck and CEFR level, select a learning mode, and start a round. Every learner signs in so progress, difficult words, preferences and study history can synchronize across devices.

1

Choose your study setup

Select one or more decks, A1–B2 levels and word types, then choose the learning path, direction and any whole-number round length. Unassigned contains words whose CEFR level has not been reviewed yet.

2

Learn and Review

Learn new words introduces only unseen words without a score. Enter any available round length manually. When the round ends, continue directly with Review or Writing using the same set. Review learned words uses only words you have already encountered and offers Remembered, Almost and Need practice.

3

Writing mode

Read the English prompt and type the German word. The dash pattern shows the expected number of letters. For nouns, choose the article. The app gives targeted feedback for articles, capitalization, umlauts, ß and small spelling mistakes.

4

Quiz and article modes

Writing uses learned words by default, with an optional All vocabulary source for experienced learners. Quiz asks you to select the translation. The der/die/das mode tests noun articles. Weak answers update the learner’s difficult-word status automatically.

5

Restart and difficult words

Use Restart round to repeat the same cards in the same order. After a round, difficult words can be tested again in Writing or Quiz mode.

6

Search and contribute

Search before suggesting vocabulary. Submit up to 20 words in one request, request a new deck, or send private feedback. Approved vocabulary contributions are counted automatically under the learner’s account name; email addresses are never shown publicly.

7

Use it like a full-screen app

Install Deutsche Vokabeltrainer on your phone or computer to open it without the normal browser address bar and navigation controls.
